🚀 프로젝트 경험

1. 빅데이터 기반 항공권 가격 변동 분석 및 예측

2023.03 ~ 2023.06

Power Automate를 활용하여 약 1년간의 스카이스캐너 항공권 데이터를 수집하고, Pandas, Scipy 등을 이용해 전처리 및 통계 분석을 수행했습니다. 계절, 요일, 시간대 등 다양한 변수가 항공권 가격에 미치는 영향을 분석하여 유의미한 패턴을 도출했습니다.

주요 역할 및 성과:

  • 데이터 수집 및 정제: Power Automate를 이용한 웹 크롤링 자동화 및 6만개 이상의 데이터 정제, 이상치 처리
  • 통계 분석: ANOVA, t-test를 통해 계절별, 요일별, 시간대별 가격 차이가 통계적으로 유의미함을 증명
  • 인사이트 도출: '오후 출발-새벽 도착' 항공권의 평균 가격이 가장 높고, '오후 출발-오전 도착'이 가장 저렴한 패턴 등 구체적인 인사이트 발굴
  • 모델링 시도: ARIMA, 선형 회귀, XGBoost 등 시계열 예측 모델링을 시도했으나, 데이터 변동성이 커 유의미한 예측 성능은 확보하지 못함 (R-squared: 0.011)
Python Pandas Scipy Matplotlib Power Automate

2. iptables 규칙 기반 DoS 공격 자동 탐지 및 방어 시스템

2022.09 ~ 2022.12

가상머신(Kali/VMware) 환경에서 다양한 유형의 DoS 공격(SYN Flooding, Land Attack 등)을 시뮬레이션하고, 이를 실시간으로 탐지하여 자동으로 iptables 방어 규칙을 추가하는 Python 기반 프로그램을 개발했습니다.

주요 역할 및 성과:

  • 실시간 패킷 캡처: Scapy 라이브러리를 사용하여 네트워크 패킷을 실시간으로 캡처 및 분석
  • 자동 방어 로직 구현: 임계값 이상의 패킷 발생 시 공격으로 간주, 공격자 IP를 추출하여 `subprocess` 모듈로 iptables 차단 규칙을 동적으로 추가하는 기능 구현
  • 성능 분석: 개발된 프로그램의 시간/공간 복잡도, 지연 시간, 자원 소모량 등을 분석하여 효율성 검증
Python Scapy Linux iptables VirtualBox

3. Next.js 기반 GitHub 프로필 명함 제작 웹 서비스

2023.09 ~ 2023.12

GitHub 계정만으로 개발자의 프로필, 사용 언어, 활동 내역 등을 요약하여 보여주는 디지털 명함을 생성하고 공유하는 웹 서비스를 개발했습니다. Next.js의 `next-auth`와 `Iron Session`을 활용하여 안전한 사용자 인증 및 세션 관리를 구현했습니다.

주요 역할 및 성과:

  • 서버사이드 렌더링(SSR): Next.js를 활용하여 SEO 최적화 및 빠른 초기 로딩 속도 확보
  • 보안 강화: GitHub OAuth 인증을 통해 사용자 정보를 안전하게 연동하고, 암호화된 쿠키 기반 세션 관리로 보안성 증대
  • 멀티플랫폼 설계: 웹 클라이언트와 함께 Android 앱 클라이언트를 고려한 통합 서비스 모델을 설계하여 확장성 확보
Next.js React next-auth GitHub API
